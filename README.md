#  Django Project

understanding django framework

## Prerequisites
```
Django 1.11.27
```

```
Python 2.7.15+
```
```
Linux(Ubuntu)
```
## Installing
Create Virtual environment
```
* sudo apt install virtualenv
* virtualenv env -p python2
* source bin/activate
* pip freeze
* pip install Django==1.11.27
* deactivate
```
